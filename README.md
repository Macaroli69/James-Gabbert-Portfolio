## Explainable AI Analysis on CIFAR-10

This project compares multiple explainable AI methods on CIFAR-10 images. The goal was to test how consistent different explanation methods are when explaining the predictions of a simple convolutional neural network.

The project uses Grad-CAM, LIME, SHAP, and Integrated Gradients. It measures explanation consistency using repeated trials, cosine similarity, and top-k intersection over union.

**Tech Used:** Python, PyTorch, Captum, LIME, SHAP, OpenCV, Matplotlib, Pandas

**Project Focus:**

- Machine learning
- Explainable AI
- CNN image classification
- Model interpretation
- Consistency testing

## Time Series Analysis of U.S. Google Search Interest in AI

This project analyzes monthly U.S. Google Trends data for AI-related search terms from 2004 through 2026. It compares general AI search interest with more specific terms such as "AI image generator" and "AI detection software."

The project also compares forecasting models such as ARIMA, ETS, and Drift to evaluate which models worked best for recent AI search interest.

**Tech Used:** R, R Markdown, fpp3, tsibble, ggplot2, ARIMA, ETS, Drift model

**Project Focus:**

- Time series analysis
- Forecasting
- Data visualization
- Model comparison
- Public interest in AI
